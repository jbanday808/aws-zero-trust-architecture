# Lessons Learned

## Overview

This project provided hands-on experience designing and deploying an enterprise AWS Zero Trust Security Architecture using AWS networking, container security, monitoring, encryption, and threat detection services.

---

## Technical Skills Gained

### AWS Networking

- Designed secure AWS VPC architecture
- Configured public and private subnets
- Implemented Internet Gateway and NAT Gateway routing
- Applied AWS Security Group best practices

### Container Security

- Built and deployed Docker containers
- Secured container workloads using ECS Fargate
- Configured Amazon ECR image repositories
- Performed vulnerability remediation using updated container images

### AWS Security Services

- Configured AWS WAF web protection
- Enabled AWS GuardDuty threat detection
- Configured AWS Security Hub monitoring
- Enabled AWS CloudTrail auditing and logging
- Implemented AWS KMS encryption

### HTTPS and Secure Access

- Configured AWS Certificate Manager (ACM)
- Enabled HTTPS/TLS encryption
- Configured secure Application Load Balancer listeners
- Implemented Route 53 DNS configuration

### Monitoring and Logging

- Monitored infrastructure using Amazon CloudWatch
- Reviewed AWS WAF block logs
- Investigated Security Hub findings
- Validated security remediation workflows
